PX4SILIntFloat结构体类. 

输出到CopterSim DLL模型的SILints和SILFloats数据
struct PX4SILIntFloat{
    int checksum;//1234567897
    int CopterID;
    int inSILInts[8];
    float inSILFLoats[20];
};

PX4SILIntFloat 类包含将在 CopterSim 模拟中使用的整数和浮点数数据数组。它支持使用提供的值列表进行初始化。
