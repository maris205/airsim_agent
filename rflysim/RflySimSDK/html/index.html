项目简介

RflySim 提供了一系列API，用户可以在仿真中以编程方式与无人系统进行交互。您可以使用这些 API 来检索图像、获取状态、控制载具等。

使用方法

PythonSDK使用方法

  • 第一步：安装python环境和依赖包（装平台）
  • 第二步：配置编辑器环境
  • 第三步：按RflySDK文档调用相关接口函数

Simulink工具箱使用方法

  • 第一步：安装Simulink工具箱（使用对应版本的MATLAB安装平台）
  • 第二步：配置自动代码生成环境
  • 第三步：按RflySDK文档使用相应模块和接口
