Speaker_Tune—蜂鸣器模块

image-20240812171336847
功能 通过这个模块可以控制蜂鸣器在特定的事件发出特定的音调。如下图所示，这个模块接受3个输入信号。
参数
输入
Tune ID l 这是可以播放的预定义“曲调”的枚举数据类型。类’PX4 TUNE ENUM’的枚举成员:
STOP TUNE
STARTUP TUNE
ERROR TUNE
NOTIFY_POSITIVE_TUNE
NOTIFY_NEUTRAL_TUNE
NOTIFY_NEGATIVE_TUNE
ARMING_WARNING_TUNE
BATTERY_WARNING_SLOW_TUNE
BATTERY_WARNING_FAST_TUNE
GPS_WARNING_TUNE
ARMING_FAILURE_TUNE
PARACHUTE_RELEASE_TUNE
IsTuneOverride 用于定义要播放的自定义曲调。用户可以使用一个常量块来定义要播放的字符串。
Trigger 这是一个触发信号，指示何时播放预定义的曲调(如果触发值变为1)或自定义曲调(如果触发值变为2)。曲调的变化将仅由该值的变化触发。
输出
