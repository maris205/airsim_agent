uORB Write—uORB消息数据发布接口模块。

image-20240809084946070
功能 该接口允许用户向uORB话题发布指定的值或结构体，通过这个模块可以向某个uORB话题发布对应的消息，话题必须经过正确的定义，一些已定义的话题放在目录C:\PX4PSP\Firmware\msg下，在生成的代码中会自动包含话题的定义文件。
参数
uORB Topic（主题名称） 输入话题名，点击按钮“Open.msg file”打开对应的消息内容，点击按钮“Open.msg folder”打开话题列表
uORB Parameter Names and Data Type区域 配置一个或多个输入端口，每个端口对应一个uORB话题的消息字段
设置变量名（输入端口名称），需要与话题的消息定义文件中的字段名一致
设置变量类型（double，single，...），需要与话题的消息定义文件中的字段名对应数据类型一致
设置变量维度，需要与话题的消息定义文件中的字段名对应数据维度一致
Sample time（s） 采样时间
输入 由uORB Parameter Names and Data Type区域配置的输入端口决定
输出
使用示例 [安装目录]\RflySimAPIs\5.RflySimFlyCtrl\0.ApiExps\6.uORB-Read-Write\Exp3_uORB_write.slx
