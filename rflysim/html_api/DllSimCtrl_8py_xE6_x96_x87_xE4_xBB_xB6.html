载具受控动态系统模型接口模块。

模块功能

它允许用户向CopterSim中的dll模型发送命令并从CopterSim中的dll模型接收数据。 为了与CopterSim进行通信，我们使用了UDP协议。 这个模块封装了UDP套接字，并提供了系列接口，用于向dll模型发送数据并接收数据。 详细描述第一部分内容。
